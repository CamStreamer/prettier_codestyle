# prettier_codestyle

CamStreamer [Prettier](https://prettier.io) config.

Template inspired by: https://github.com/azz/prettier-config

## Usage

**Install**:

```bash
$ yarn add --dev @camstreamer/prettier-config
```

**Edit `package.json`**:

```jsonc
{
  // ...
  "prettier": "@camstreamer/prettier-config"
}
```

## Publishing

```
np --no-tests --no-2fa
```
