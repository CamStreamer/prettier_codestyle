# prettier_codestyle

Our Netrex [Prettier](https://prettier.io) config.

Template stolen from here: https://github.com/azz/prettier-config

## Usage

**Install**:

```bash
$ yarn add --dev @netrex/prettier-config
```

**Edit `package.json`**:

```jsonc
{
  // ...
  "prettier": "@netrex/prettier-config"
}
```
